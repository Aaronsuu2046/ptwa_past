# 遊戲介紹

## Grad3-4 重量的量感與估測（weight_sense）
[![weight_sense](https://img.shields.io/badge/weight_sense-v0.1.1-blue.svg)](./weight_sense)

- 遊戲路徑：進入[遊戲網站](https://PTWA-NPO.github.io/)後，點擊左方列表中的**三年級**，選擇**重量的量感與估測**

### 遊戲簡介

根據題目，選擇直角鈍角銳角，正確即可過關。
